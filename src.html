My Janky Home Lab

It all began when I stumbled across a NetworkChuck youtube video titled You need a NAS RIGHT NOW!!
Not knowing what a NAS is but recognizing some sort of storage device, thanks to LinusTechTips Petabyte Project,
I watched the video. After some quick research, I found the OS TrueNas Core which is a Freebsd-based operating
system that is totally free and allows for NAS configuration of normal computer hardware. Long story short I
used old hardware I had laying around, installed FreeNAS on a thumb drive, and booted it on the spare parts computer.

The Server Setup

Initially, I did not realize that raid pools could only be created with drives of the same size. After a small hiccup and
some help installing 3 3TB drives from my brother while I was at college, I was ready to configure the server. I have 1
pool with the 3TB drives running raidz 1. So, in total, I have 6TB of useable data. I have one more drive-by itself for
less-import data such as the Minecraft server I host. The big data pool stores personal data for my family, and is used for
NextCloud plus a Plex server. Along with the NAS in the storage room, I have a network switch, small UPS, a monitor to
monitor the server locally, and a Raspberry Pi 4.
